# Personal Finance Analytics Dashboard

## Project Overview
This project demonstrates an end to end finance analytics workflow using PostgreSQL for data processing and Tableau for data visualisation.

The goal is to analyse income, expenses, and savings behaviour over time and present actionable insights through an interactive dashboard.

---

## Tools Used
- PostgreSQL for data cleaning, metrics, and views
- SQL for data transformation and aggregation
- Tableau for dashboard creation and visualisation
- GitHub for version control and portfolio presentation

---

## Dataset
Synthetic personal finance transaction data containing:
- Transaction dates
- Income and expense amounts
- Spending categories
- Monthly savings calculation

The dataset was generated for portfolio demonstration purposes.

---

## SQL Workflow
1. Data cleaning and standardisation
2. Metric calculations such as total income, total expenses, and savings
3. Creation of reusable SQL views for Tableau consumption

---

## Tableau Dashboard
The dashboard includes:
- KPI cards for total income, expenses, and savings
- Income vs expenses trend over time
- Spending breakdown by category
- Monthly savings trend
- Interactive date range and category filters

---

## Key Insights
- Identification of high spending categories
- Monthly savings trends and negative savings periods
- Income and expense alignment over time
